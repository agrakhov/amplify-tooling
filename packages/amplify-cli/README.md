# AMPLIFY CLI

The AMPLIFY CLI is the unified CLI for the Axway AMPLIFY platform.

## Installation

	npm i -g @axway/amplify-cli

## Quick Start

Show all available commands:

	amplify

Install a package:

	amplify pm install appcd

Log into the Axway platform:

	amplify auth login
